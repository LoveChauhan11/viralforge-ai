# Command: Create Handoff

Produce a self-contained sprint/task handoff:

- scope and backlog IDs completed/not completed;
- branch/commit/environment and deploy state;
- working vertical flows;
- contracts, migrations, ADRs and configuration changes;
- tests/commands/results and evidence;
- security/privacy/rights/accessibility/cost review;
- known issues, risks, blockers, owner decisions;
- exact rollback and exact next task/entry criteria.

Update all memory files and traceability before handoff. Do not say “done” when a gate is missing, skipped, or only locally assumed.
