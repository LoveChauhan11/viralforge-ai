# Command: Start Sprint

Input: sprint number.

1. Read `AGENTS.md`, Cursor rule, all memory-bank files, roadmap, target backlog, Definition of Done, linked contracts and prior handoff.
2. Verify prior sprint exit evidence and clean/understood worktree. Stop on unexplained changes or missing gate.
3. Update `ACTIVE_TASK.md` with sprint goal and first backlog ID.
4. Produce dependency graph, bounded parallel lanes, owners/agents, risks, ADRs, migrations, contract changes, test/evidence plan.
5. Confirm fake-provider path and no credential blocker.
6. Begin only the earliest unblocked task; do not silently widen scope.
